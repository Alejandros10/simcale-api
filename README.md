# Simacel-App

## Requisitos previos

1. **Node.js**  
    Tener instalado [Node.js](https://nodejs.org/en).  

2. **Angular CLI**  
    `npm install -g @angular/cli`

3. **Clonar repositorio**
    `git clone https://github.com/Alejandros10/simcale-app.git`

4. **Instalar dependencias**
   `npm install`

5. **Levantar el servicio**
    `ng serve`
